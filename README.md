# Instruction
A little knowladge of scripting is needed if you want to use this. Or if you just want the theme, copy the `.termux` folder and replace it with your current one.
- If you want wallpaper and true italic font see [termux-monet](https://github.com/KitsunedFox/termux-monet).
- I'm using [termux-pacman](https://github.com/termux-pacman/termux-packages/) check [here](https://wiki.termux.com/wiki/Switching_package_manager) if you want to switch from apt.
